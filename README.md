# 🧪 もしも宇宙ラボ

**こわして学ぶ、宇宙の実験室。**

もしも太陽が消えたら? 木星が恒星になったら? 地球が公転をやめたら?
教科書は「正しい宇宙」しか見せてくれない。このアプリは「もしも」を自分の指で実験できる。

🔗 **https://syoudai0514.github.io/moshimo-space-lab/**

## 特徴

- **本物の物理**: 太陽を含む全天体が互いに重力を及ぼし合う N体シミュレーション(リープフロッグ法 + 適応刻み幅)。惑星が弾き飛ばされるのも太陽に落ちるのも、すべて万有引力の計算結果
- **🧪 もしも実験**: 「太陽が消えたら」「木星が恒星になったら」など、ワンタップで始まる思考実験シナリオ。それぞれに物理の解説付き
- **💡 イベント解説**: 惑星が太陽に飲み込まれたり宇宙の彼方へ飛んでいったりしたら、通知をタップすると「なぜそうなったか」の解説が読める
- **📸 結果シェア**: 実験結果をカード画像にしてSNSでシェア
- **自由実験**: 惑星をドラッグで動かす、重さを1000倍にする、太陽と場所を入れ替える…なんでもあり
- **🔭 観察モード**: 実験の舞台を眺める資料室。銀河系(±30億年の回転)と宇宙の歴史(ビッグバン〜238億年)。原子が生まれる前の時代は 🔬 で同じ時間軸のままミクロの世界にズームインできる

## 開発(Web 版)

ビルド不要の静的サイト(Three.js 同梱)。

```bash
python3 -m http.server 8000
# → http://localhost:8000
```

## Android アプリ版(Capacitor)

Web アプリを Capacitor で Android アプリ化しています。
ビルド・署名・Google Play 提出の手順は **[RELEASE.md](./RELEASE.md)** を参照。

```bash
npm install
npm run sync          # www/ を生成し Android プロジェクトに反映
npx cap open android  # Android Studio で開く
```

- 広告(AdMob)などネイティブ専用の処理は `src/native/main.js`(Web 版では読み込まれない)
- 現在は Google 公式の**テスト広告 ID**。本番 ID への差し替え方は RELEASE.md の手順2
- ストア掲載文・素材は `store/`、プライバシーポリシーは `privacy.html`
